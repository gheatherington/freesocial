# 01-02 Intervention UX Copy

## Blocked Feed Behavior

- "Session paused: take a 30-second break before loading more."
- "Infinite scrolling is disabled in FreeSocial by design."

## Unsupported Action Message

- "This action is not supported in FreeSocial yet."
- "Use the official app for this action."

## Limitation Disclosure

- "FreeSocial is a controlled companion, not a full replacement app."
- "Some social features depend on provider-approved API access."

## Escalation Message

- "You have reached your current session limit. Next unlock available after cooldown."
- "Repeated bypass attempts increase cooldown duration."

## Requirement Mapping

- CC-03: direct intervention messages
- POL-03: plain-language limitations
